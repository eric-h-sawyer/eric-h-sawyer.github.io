<!DOCTYPE html>
<html>
  <head>
    <title></title>
  </head>
  <body>
  </body>
</html>
  • The document type declaration <!DOCTYPE html> must be the first line of code in your HTML document.
  • The <html> element, the root of an HTML document, should be added next. All HTML code should be contained between the opening and closing <html> tags.
  • The content inside the <body> element is what is displayed to the screen.
  • The <head> element goes above the <body> element and contains the metadata for the web page.
